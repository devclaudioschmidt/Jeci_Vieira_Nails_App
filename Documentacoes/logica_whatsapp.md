Lógica de comunicacao com o cliente através de Whatsapp

Regras para comunicaçao.

1. Cliente agenda horario, ao concluir seu agendamento sistema envia msgm pré programada para o admin através do whatsapp (telefone definido em configuraçoes) confirmando seu agendamento.

2. Cliente reagenda seu horario, no final do ciclo envia msg ao adm via Whatsapp informando sobre o que foi feito.

3. Cliente Cancelar horário, envia msg para adm via Whatsapp informando cancelamento.


4. ADM cancela horário de cliente, envia msg para cliente informando cancelamento.

5. ADM reagenda horário, de cliente, envia msg para cliente informando reagendamento.
